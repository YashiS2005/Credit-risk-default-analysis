# Credit-risk-default-analysis
Power BI dashboard analyzing loan default patterns across borrower segments
## Credit Risk & Default Analysis Dashboard

### Objective
Analyze loan default patterns across borrower segments to identify high-risk profiles.

### Tools Used
- Power BI
- Excel

### Key Insights
- Higher default rates in lower loan grades
- Renters show higher default risk than homeowners
- Certain loan intents consistently default more

### Files
- Credit_Risk_Dashboard.pdf – dashboard snapshot
